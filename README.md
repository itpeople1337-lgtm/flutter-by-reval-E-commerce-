# Point of Sales System (POS) & Stock Management

This is a comprehensive Point of Sales (POS) and Stock Management System, built with Flutter and Dart, designed to help businesses efficiently manage their products, sales, and inventory. The app supports multiple platforms including Android, iOS, Windows, and Mac.

## Features

### Product Management
- **Add/Edit Product Information** with images.
- **Product Stock Management** including stock counts and stock movement.
- **Search Products** by name or code.
- **Top Sales Items** based on performance.

### Customer and Supplier Management
- **Add/Edit Customer and Supplier Information**.
- **Search Suppliers** and **Customers**.
- **Create Debts** with date and client data (Call, SMS, WhatsApp, Email).

### Sales and Transactions
- **Add/Edit Incoming and Outgoing Information**.
- **Generate PDF Receipts** for sales and transactions.
- **Print PDF Files** or open them in external PDF readers.
- **Export Sales Data** to Excel files (XLS).
  
### Analytics and Reporting
- **Top Sales Customers**: Track your most valuable customers.
- **Analytics with Bar/Line Charts** to visualize data.
- **Calculate Profits/Losses** from sales, expenses, and debts.
- **Multiple Reports**: Transactions, Expenses, Debts, Stock Movement, Sales Monthly (client, item, date), and Profits/Losses.

### Expense Management
- **Create Expense Lists** with payment methods.
- **Add/Edit Credit Card Information** for easy payment tracking.
- **Tax and Discount System** for sales.

### Multi-Language and Multi-Platform Support
- **Multiple Languages Supported**: English, French, Spanish, Portuguese, and Arabic.
- **Cross-Platform Support**: Android, iOS, Windows, and Mac.

### Communication Features
- **Call, SMS, WhatsApp, Email Integration** for quick customer communication.
  
### User Management
- **User Profile Data** for personalized experiences.
- **Register/Sign in** with Email, Google, or Facebook.

### User Interface
- **Responsive Design**: Adaptive to different screen sizes.
- **Drawer Navigation**: Smooth and intuitive navigation.
- **Animations**: Attractive and smooth transitions throughout the app.

### Company and Store Management
- **Create Stores** and manage multiple store locations.
- **Add Company Info** such as Logo and Signature for invoices.
  
### Export Data
- **Export Product Data** into Excel.
- **Export Customer and Supplier Data** to Excel.
- **Export All Expense Data** to Excel.

### Image Management
- **Pick Up Image** from Gallery or Camera for products and receipts.

### Customization
- **Dark/Light Theme** toggle for user preference.
  
### Compatibility
- **Android 12 and Newer** compatible.

## Tech Stack
- **Development IDE**: Android Studio, VS Code.
- **Programming Language**: Flutter (Dart).

## Setup

### Prerequisites
- Flutter SDK
- Dart SDK
- Android Studio / VS Code

### Getting Started
1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/point-of-sales-system.git
    ```

2. Install dependencies:
    ```bash
    flutter pub get
    ```

3. Run the application:
    ```bash
    flutter run
    ```
